지라 commit message 연동
git commit -m "#이슈번호 내용"
ex) git commit -m "#S08P12C106-48 commit message"
이렇게 커밋하면 48번 이슈에 커밋메시지 연동됨

branch 간단 커맨드
1. 현재 내가 위치한 branch 확인( *가 붙은게 현재 활성화 branch )
git branch

2. 원격 저장소의 branch 확인
git branch -r

3. branch 생성
git branch 브랜치명
ex) git branch test

4. 생성한 branch로 이동
git switch 브랜치명
ex) git switch test
